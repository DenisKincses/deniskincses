Scratch Extension - Made by DenisKincses

(Some of these blocks ONLY work with Electron.)

  • windtools.js

    Blocks that can change the X and Y position of the window, and change its height and width. Try it out here!

    Opens URLS on the default Browser. Try it out here!

  • alert.js

    Alert windows that popup. Try it out here!

  • getinfo.js

    It gets your browser, and OS information, into a reporter. Try it out here!

  • geolocation.js

    A failed attempt to making a Scratch Extension that uses the Geolocation API. Try it out here!

  • embedpage.js
